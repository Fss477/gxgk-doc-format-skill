# 广西国控集团公文格式排版技能

根据广西国控资本运营集团有限责任公司公文格式规范（2025 年 7 月版）自动排版 Word 文档。

## 触发条件

用户要求：
- 按广西国控集团格式排版公文
- 格式化 Word 文档为集团标准
- 公文格式修改
- 发送 Word 前按集团规范排版

## 格式规范

### 页面设置
- 页边距：上 37mm，下 35mm，左 28mm，右 26mm
- 页码：宋体，小四号（14pt），居中，格式为"-1-"

### 标题格式
- 公文标题：**方正小标宋简体**，二号（22pt），居中，不加粗
- 日期行：居中，16pt

### 正文格式
- 正文字体：**仿宋_GB2312**，三号（16pt）
- 对齐方式：两端对齐
- 首行缩进：2 个中文字符
- 行距：固定值**29pt**
- 段前段后：0 行

### 标题层级
- 一级标题（一、）：**黑体**，三号（16pt），不加粗，行距**33pt**
- 二级标题（（一））：**楷体_GB2312**，三号（16pt），不加粗，行距**33pt**
- 三级标题（1.）：**仿宋_GB2312**，三号（16pt），加粗，行距**33pt**
- 四级标题（（1））：**仿宋_GB2312**，三号（16pt），不加粗，行距**33pt**

### 数字格式
- 文中的阿拉伯数字：**Times New Roman**

### 表格格式
- 表格标题：方正小标宋简体，小二
- 表头：黑体，小四（14pt），居中
- 表格内容：仿宋_GB2312，小四（14pt）
- 边框：完整可见

### 特殊段落
- 主送单位：仿宋_GB2312，16pt，左对齐，顶格
- 附件说明：仿宋_GB2312，16pt，"附件"二字加粗
- 发文单位：仿宋_GB2312，16pt，右对齐
- 成文日期：仿宋_GB2312，16pt，右对齐
- 联系人信息：楷体_GB2312，小四（14pt），左对齐

```

```

## 依赖

- Python 3.x
- python-docx

## 注意事项

1. 如缺少所需字体（方正小标宋简体、仿宋_GB2312、楷体_GB2312 等），会返回缺失字体列表
2. 文档中的空行保留，但会统一格式
3. 附件标识从"附件"或"附件 1"开始特殊处理
4. 日期格式如"（2025 年 4 月 19 日）"居中处理
